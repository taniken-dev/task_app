・プロジェクト名；task_app

Flaskで作ったタスク管理アプリです。
ユーザー登録・ログインができて、タスクの追加・更新・削除ができます。

大学の授業でベースのコードをもとに、グループで使ってみて気になったところを
話し合いながら改善しました。

・ポートフォリオサイト
https://sites.google.com/chibatech.ac.jp/24g2079/1年後期/認知情報科学演習


・使用技術

- Python / Flask
- Flask-SQLAlchemy（DB操作）
- Flask-Login（認証）
- Flask-Migrate（マイグレーション）
- PyMySQL / Docker

・ローカルで動かす場合

git clone https://github.com/taniken-dev/task_app.git
cd task_app
docker-compose up --build

http://localhost:5000 にアクセスすると使えます。


・グループで改善した点

実際に動かしてみて、使いにくいと感じた箇所を話し合って直しました。

1. 新規登録画面にログインページへ戻るボタンがなかったので追加した。
2. タスク作成時に名前や締切日を空のまま送るとエラーになっていたので、フラッシュメッセージで案内するようにした。
3. タスクの作成日時がUTCで記録されていて9時間ズレていたので日本時間に直した。
4. アカウントを削除する手段がなかったので削除ページを新しく作った。
5. 削除ボタンを押すと即座に消えてしまっていたので、確認のポップアップを出すようにした。


・気になっている課題

- データベースにSQLiteを使っているのでRenderを再起動するとデータが消える
- タスクの編集・締切管理など、まだ追加したい機能がある
